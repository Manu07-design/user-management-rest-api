#User Management REST API

A simple REST API project to understand backend fundamentals.

## Features
- Create user
- Get all users
- Get user by ID
- Update user
- Delete user

#API Endpoints

| POST | /users | Create user |
| GET | /users | Get all users |
| GET | /users/{id} | Get user |
| PUT | /users/{id} | Update user |
| DELETE | /users/{id} | Delete user |

#Tech Stack
- REST API
- Python
- Postman (for testing)
